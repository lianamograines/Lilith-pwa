Лилит — Пески и Тени (PWA)
=================================

Готовое минимальное веб‑приложение (PWA) с интерактивной историей «как Клуб Романтики», только текст + выборы.

Файлы:
- index.html — разметка, meta для iOS, подключение скриптов
- style.css — оформление
- app.js — движок выбора + сюжет
- manifest.webmanifest — PWA‑манифест
- sw.js — service worker для офлайн‑работы
- icon-192.png, icon-512.png — иконки приложения

Как развернуть С ТЕЛЕФОНА (iPhone):
1) Открой любой хостинг со статикой, который принимает загрузку архива с телефона (например, «Netlify Drop», «Glitch»).
2) Загрузите архив lilit-pwa.zip. Сервис даст вам HTTPS‑ссылку.
3) Откройте ссылку в Safari → кнопка «Поделиться» → «На экран «Домой»».
4) Запустите иконку. Первый запуск потребует интернет, затем приложение работает офлайн.

Примечания iOS:
- Добавлен meta apple‑mobile‑web‑app‑capable и манифест. Даже если иконки/манифест игнорируются, добавление на Домой работает через Safari.
- Service Worker активируется после первого открытия. Для обновления версии просто перезалейте файлы (или измените версию кэша в sw.js).

Настройка сюжета:
- Правьте объект story в app.js (узлы, текст, выборы).
- Можно добавлять действия (action) и флаги в объект flags для логики.
